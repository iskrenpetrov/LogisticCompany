Логистична компания

Логистична компания е проект за Нов Български Университет, разработен със Spring Framework. Spring е базиран на програмния език Java. Използван е Thymeleaf, който е template engine, за да реализираме front-end частта на проекта.

Github Repository

Проектът може да се разгледа в платформата Github на този адрес.
